Este repositório contém as atividades realizadas durante as aulas de Algoriitmos e logica da programação.
As tarefas foram realizadas para abordar conceitos de lógica, portugol, cálculo, etc.
E envolvem resolução de problemas, desenvolvimento de código, projetos colaborativos, etc.

Antes de mais nada, gostaria de pedir desculpas pela personalidade excessiva que coloquei em cada uma das tarefas!
Sei que algumas partes podem ter ficado um pouco mais criativas (ou excêntricas) do que o necessário, mas acredite, foi tudo feito com a melhor das intenções!

Sei que a atividade tem um objetivo acadêmico, mas não pude evitar dar um pequeno "toque de autenticidade" em cada parte.
Por favor, não leve isso a mal!

O objetivo geral desta atividade foi aplicar os conceitos aprendidos em aula de forma prática e criativa.

Como Executar as Tarefas:
Clone este repositório para sua máquina local.
Navegue até a pasta da tarefa que deseja visualizar.
Abra os arquivos no visuAlg para revisar ou executar as soluções.
